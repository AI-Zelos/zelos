API Reference

    
REST API Endpoints (HTTP Adapter on port 9876):

GET  /api/v1/health              — Runtime health
GET  /api/v1/metrics             — Runtime metrics
POST /api/v1/goals               — Submit a new goal
GET  /api/v1/goals               — List all goals
GET  /api/v1/goals/{goal_id}     — Get goal status
DELETE /api/v1/goals/{goal_id}   — Cancel a goal
GET  /api/v1/agents              — List registered agents
GET  /api/v1/agents/{agent_id}   — Get agent details
GET  /api/v1/audit               — Query audit log (admin)
GET  /api/v1/tenants             — List tenants
GET  /api/v1/cluster             — Cluster status
GET  /api/v1/approvals/pending   — Pending HITL approvals
POST /api/v1/approvals/{id}/approve  — Approve
POST /api/v1/approvals/{id}/reject   — Reject

Auth: Bearer {api_key} header or auth_context in request body
